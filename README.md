# Librarian

A TUI for managing Johnny Decimal vaults in Obsidian.

## Install

```bash
make install
```

## Usage

```bash
librarian
```

## Keys

| Key | Action |
|-----|--------|
| `j/k` | Navigate |
| `h/l` | Collapse/Expand |
| `enter` | Open README / Toggle |
| `y` | Copy ID |
| `n` | New item |
| `a` | Archive |
| `/` | Search |
| `q` | Quit |
