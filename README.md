# Libraio

A TUI for managing Johnny Decimal vaults in Obsidian.

## Install

```bash
make install
```

## Usage

```bash
libraio
```

## Keys

| Key | Action |
|-----|--------|
| `j/k` | Navigate |
| `space` | Toggle / Open |
| `y` | Copy ID |
| `n` | New item |
| `a` | Archive |
| `m` | Move item |
| `/` | Search |
| `?` | Help |
| `q` | Quit |
